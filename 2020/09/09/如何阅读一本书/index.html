如何阅读一本书
发表于 2020-09-09| 更新于 2020-09-09

[TOC]

阅读的层次

阅读的活力与艺术

本书是为想从文字中获取信息的人所写。新媒体因为过多的信息阻碍了人们的理解。

阅读是一件主动的事情，越主动获得越多。主动+技巧=高效

对作者说的话很了解，就增加资讯，但不一定增加理解。书中的东西超出你了解的范畴时理解力才能提升。阅读不能借助他人的解释。本书主要谈论如何增进理解力。

要被启发，不仅需要知道作者所说的话，还要明白为什么作者要这样说。指导型的学习与自我发现型的学习区别在于教材的不同，指导型的学习依赖规定的内容，自我发现型则立足于自然或世界。思考在两者中都有，但 不仅要思考还要运用想象力。

阅读相当于向一位缺席的老师学习。由于没有老师的帮助，懂得如何让书本来教导我们很重要。

阅读的层次

  1. 基础阅读(elementary reading)：这个句子在说什么？我们需要知道作者想表达什么意思，这个层次的问题解决之后可以提升阅读速度。
  2. 检视阅读(inspectional reading)：这本书大体上在谈论什么？这本书包含哪些部分？这是哪类书——小说、历史、论文？需要重视它的价值，否则会导致你详读本该略读的书。
  3. 分析阅读(analytical reading)：全盘的，完整的阅读。读者会提出许多系统性的问题。
  4. 主题阅读(syntopical reading)：读者会读很多书，并找出这些书的相关性，指出一个所有书都谈到的主题。最后能提出一个在所有书中都没提过的主题分析。

阅读的第一个层次：基础阅读

1970年代是美国的“读书的年代”。第一是因为美国在继续推行全民教育，第二是阅读教育本身起了变化（ABC\(\rightarrow\)发音法 | 视觉法 etc.）

学习阅读的阶段

  1. 阅读准备阶段(reading readiness)：出生\(\rightarrow\)六七岁。身体方面的准备(视力、听力)、精神方面的准备(与其他人一起学习的能力，注意力)
  2. 简单读物阶段：能够看懂符号所代表的意思，阅读简单的读物
  3. 快速建立词汇阶段：通过上下文的线索学会不熟悉的词汇
  4. 精炼增进阶段：青少年时期达到。对其前面的技巧进行提升

上述阅读的阶段都属于基础阅读。很多人没有基础阅读的能力，但是阅读本身并不是一件困难的事情。

阅读的第二个层次：检视阅读

在进入这个层次之前，首先要保证精通基础阅读。不必查生字的意思，也不会被文法或文章结构阻碍。检视阅读有两种：有系统的略读和粗浅的阅读。

有系统的略读

目标是发现这本书值不值得多花时间仔细阅读。

  1. 书名页\(\rightarrow\)序 了解作者写这本书的特殊角度，归类这本书
  2. 目录页 对架构作概括性的理解
  3. 索引 ？？？没看太懂
  4. 出版者的介绍 到第4步为止就可以判断是否需要继续读下去
  5. 挑选几个目录中看起来和主题相关的篇章来看
  6. 随意翻一下，读几段，但不要太多。不要忽视最后结尾的两三页

粗浅的阅读

头一次面对一本难度的书的时候，从头到尾先读完一遍，碰到不懂的地方不要停下来查询或思索。需要注意你能理解的部分。

逗留与倒退导致阅读速度变慢。可以用手作为指针，手的速度要比眼睛要快一点，强迫眼睛跟着手移动。

慢不该慢到不值得，快不该快到有损于满足于理解。

主动阅读时需要以适当的顺序提出适当的问题：

  1. 整体来说，这本书在谈论什么？找出本书的主题，并且了解作者如何一次发展这个主题，逐步从核心主题分解出从属的关键议题
  2. 局部是这么说的？找出主要的想法、声明与论点
  3. 前两个问题回到之后，要考虑这本书说得有道理码？是哪部分有道理？光了解作者的想法是不够的
  4. 本书与你有什么关系？书中提供的资讯有什么意义，为什么作者认为知道这件事很重要？自己有必要去了解吗？ ？？？啥叫找出其他相关的、更深的含义或建议，没说咋找啊

做笔记是不可或缺的。能保持清醒，将思考通过语言表现出来，还可以将感想写下来保住记住作者的思想

检视阅读时应该做结构笔记(structural note-making)记下全书的架构而不是细节内容

概念笔记(conceptual note-making)记下的是作者的观点，当深入阅读后也会出现自己的观点。

还有辨证笔记(dialectical note-making)，就一个主题把所有的陈述和疑问都列出来。

阅读的第三个层次：分析阅读

一本书的分类

本书提到的方式也适用于那些较为简单的读物，例如篇幅较短的文章。

第一阶段的第一规则：知道自己在读的书的种类很重要。

很多人会忽略书名或序言，因为他们认为对书进行分类是毫无必要的。

分类规则

理论性的作品是在教你这是什么，实用性的作品 在教你如何去做你想要做的事，或你认为应该做的事。

理论性作品的分类：

历史就是在谈论发生在特定时间的、特定地点的真实事件。科学不会在意时间、地点，它更加关注普遍规律。

科学提到的内容普通人很难在日常生活中进行查证，但是哲学提出的事实或观察一般不会超越一般人的生活经验。

如果我们要跟随书本的指导，就应该学习用适当的态度来阅读不同的书。

透视一本书

任何一本值得读的书，都会有一个整体性与组织架构。

第一阶段的第二规则：使用一个单一的句子，或最多几句话来叙述整本书的内容。

每本书都有一个想要表达的主题，整本书会围绕这个主题展开。可以有自己的诠释，但是也不能爱怎么说就怎么说。该规则强调书籍的整体性。

第一阶段的第三规则：将书中重要篇章列举出来，说明它们如何按照顺序组成一个整体的架构

每个书都是由不同的部分组成的，因此我们需要弄清楚它们之间的关系，也就是它们如何构成整体的架构。该规则强调书籍的复杂性。

纲要不一定要与 作者完全一致，作者的纲要可能只是表象，而读者要做的是透过现象看本质。

规则三是规则二的必要条件

阅读与写作是相辅相成的。传达知识的书的各个部分可以是独立的，但是文学作品却不能。

第一阶段的第四规则：找出作者要问的问题。一本书的作者在开始写作时，都是有一个问题或者一连串的问题，而作者写的这本书的内容就是一个答案或者许多答案。

理论性问题：

  • 某件事存在吗？在什么样的情况下存在？
  • 是什么样的事？
  • 发生的原因是什么？
  • 这件事的目的是什么？
  • 这件事造成的影响是什么？
  • 这件事有什么特征？
  • 与不相同的事件的关联是什么？
  • 这件事如何进行的？

实用性问题：

  • 有哪些结果可以选择？
  • 应该采取什么样的手段才能获得某种结果？
  • 要采取哪些行动，以什么顺序采取行动？
  • 在什么样的条件下，这样做会比较好一点？

与作者找出共通的词义

第二阶段的第一规则：找出共通的词义。必须抓住书中重要的单词，搞清楚作者是如何使用这些单字的，即单字的精确意义。

因为语言是有模糊地带的，具有多义性，因此需要先解决语言的问题，然后再超越语言，处理语言背后的思想涵义。

不需要关注每一个字，只需要关注哪些更为重要的字即可。对于读者而言，最重要的字就是那些你不懂的，让你头痛的字，有些字可能作者认为很重要，但是你已经掌握，因此你不会觉得头痛。此时在这种已经掌握的字上，就已经与作者达成了共识。

当你了解某一领域的时候可以容易分辨出专业术语，当你不了解时，那些不平常的单字就是专业术语。

当发现一位作者告诉你某个特定的字曾经被其他人如何使用，而它为什么选择不同的用法时，这个字就是重要的。

找出的某一个关键字，在书中可能代表单一的词义，也可能代表不同的词义。

假如多重含义，要看这些意义之间的关系如何，并且明确在某个地方使用的是哪种含义。并且最后了解作者为什么要更换含义。

简单的说字与词组 与 意思之间是一种多对多的关系，我们要理解的是背后的意思而不是仅仅关注作者的用字用词。

判断作者的主旨

有的句子并不直接表达主旨而只会提及一个问题，而本书的主旨就是那个问题的答案。

句子与主旨也是多对多的关系，例如一个复合句就可能表达多个主旨。

读者读起来吃力的地方只是可能是重要的，但是也不一定，但是哪些读者觉得难懂的地方是应该更加仔细的理解的。作者认为重要的句子应该是那些他所下的肯定或否定的判断。有些作者会把重要的句子用记号标出，有些会将主旨写在最前面。

怀疑是智慧的开始

前面判断的关键字所在的句子很有可能是表达主旨的句子。

判断是否理解作者主旨的最好方式就是用自己的话表达该主旨的意思。而且假如能给出例证，对于这一主旨的理解就更加深刻了。拒绝口语主义，不要仅仅去玩弄文字

主旨可能由归纳或演绎来得出

公正地判断一本书

受教不是卑躬屈膝，即使作者的水平要高于读者，读者也应该有自己的见解，因为没有一本书是好到无懈可击的。

在你说出“我同意”，“我不同意”，或“我暂缓评论”之前，你一定要能肯定地说：“我了解了。”有时候可能确实没读懂，这时候做出我不懂的判断也不是什么坏事，在没懂的时候就应该心平气和的从头研究这本书，除非这本书实在结构混乱让人难以读懂。而且不要盲目针对某一作者的一本书去评价，因为你的反对理由很有可能在同一个作者的其他书籍中找到。

我们所知道的知识都有理论支持，或至少有一些证据在支持着，但我们不知道什么时候会出现新的证据，或许就会推翻我们现在相信的事实

辅助阅读

一本书困扰住你时，我们也不会建议你去阅读评论这本书的文章。整体来说，在你找寻外力帮助之前，最好能自己一个人阅读。如果你经常这么做，最后你会发现越来越不需要外界的助力了。

外在阅读的来源：第一，相关经验。第二，其他的书。第三，导论与摘要。第四，工具书。

个人觉得至少要完整的阅读一遍之后再借助辅助阅读的帮助。

假如你能举出与书中观点类似的例子，那么就大致的了解的这个观点的重点。

依照写作的时间顺序来读，对你了解最后写的作品有很大帮助。

伟大的作者也是伟大的读者，想要了解他们，不妨读一读他们在读的书。

在工具书中你只能看到约定俗成的观念。

阅读不同读物的方法

如何阅读实用型的书

实用型的书只能提供建议，但是实际解决问题时还是需要依靠读者自己。

在赞同了一本实用型的书之后需要按照作者提议的来行动。假如不愿意行动的话，可能并不是真正的同意了。

如何阅读想象文学

论说型作品要传达的是一种知识，但是想象文学要阐述一个经验本身。

在想像文学中，不要去找共识、主旨或论述。那是逻辑的，不是诗的，二者完全不同。

阅读故事、戏剧与诗的一些建议

在阅读故事的时候，应该快读、全心全意的读，最好能一口气读完。要尽量试着活在故事的世界里，这样能够更好的理解这个故事。

如何阅读历史书

真正发生了什么是很难确认的，所以历史学家所写的历史也很难保证是完整的事实。

修昔底德说过，他写历史的原因是：希望经由他所观察到的错误，以及他个人受到的灾难与国家所受到的苦楚，将来的人们不会重蹈覆辙。

如何阅读科学与数学

科学论文是写给特定领域中的读者的，在其他的领域也有这一现象，但是科学领域中这一现象更严重。

如何阅读哲学书

孩子喜欢询问为什么，而成人更加关心事情是否如此。

哲学家提出的问题并不复杂， 但是要回答确实困难的。

两种主要不同的哲学领域：存在与变化、善与恶。

可以去读一下正在读这本书的作者所读的书，这样能够有更好的理解。

如何阅读社会科学

社会科学的作品通常不会进行用语说明。也就是有些定义可能是存在歧义的。

在阅读社会科学时，关于一个主题通常要读好几本书，而不会只读一本书。

阅读的最终目标

阅读的第四个层次：主题阅读

在做主题阅读时，会出现一种很矛盾的现象。虽然这个层次的阅读被定义为就同一个主题，阅读两种以上的书，意思也是指在阅读开始之前，这个主题就已经被确认了，但是换个角度来说，这个主题也是跟着阅读走的，而不是事前就能定出来的。以爱这个例子来说，在你决定自己要读些什么之前，你可能已经读了好几百本相关的著作了。等你都读完之后，你会发现有一半的书其实跟主题根本无关。

检视阅读：能够让你对自己想要研究的主题有个清晰的概念；简化你的书目到一个合理的程度。

读者应该不预设立场。

  1. 准备书单并浏览，确定哪些与主题有关，帮助自己对主题建立起清楚的概念。
  2. 找出这些书中与主题最相关的章节
  3. 根据主题创建一套中立的词汇，来引导作者与自己达成共识
  4. 建立一个中立的主旨，并列出一系列的问题来提供给作者回答。
  5. 界定主要以及次要的议题
  6. 对以上形成的讨论进行分析

阅读与心智的成长

我们并不希望很权威地告诉你，哪些书对你来说是伟大的作品。我们的重点是，你该自己去找出对你有特殊价值的书来。这样的书能教你很多关于阅读与生命的事情。这样的书你会想一读再读。这也是会帮助你不断成长的书。
